Cómo editar la tabla

Solo tienes que cambiar casas-cambio.json. El HTML ya está preparado para leer el contenido y reordenar la tabla automáticamente.

1. Cambia los enlaces Sustituye cada ?ref=TU_CODIGO por tu referido real.
2. Reordena casas Usa sortOrder o sortOrderByCurrency.
3. Ajusta tarifas Edita rateByCurrency y feeByCurrency.

La app funciona bien en GitHub Pages porque solo necesita este index.html y el JSON en la misma carpeta.
